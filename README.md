# Electronic supplements

This folder contains the executable Jupyter notebook and a static browser-readable rendering of the same material.

GitHub sometimes fails to render Jupyter notebooks in the web preview and shows only `An error occurred`. If that happens, use the HTML version instead:

- `Agricola_Henkel_Calculation_Spectrum.html`: static HTML rendering with code cells and recorded outputs in notebook order. The plots are embedded directly in the HTML file.
- `Agricola, Henkel Calculation Spectrum.ipynb`: original executable Jupyter notebook.

To view the HTML file from GitHub, download it and open it in a browser.
